ค้นหาข้อมูลนักเรียนอย่างรวดเร็ว

ป้อน รหัสประจำตัว, ชื่อจริง, นามสกุล, ชื่อเล่น หรือเบอร์โทร เพื่อดึงหน้าต่างประวัติทันที

ศูนย์จัดการฐานข้อมูลคลาวด์ Supabase Real-time

ระบบฐานข้อมูลได้รับการอัปเกรดเป็น **Supabase Cloud (PostgreSQL)** เรียบร้อยแล้ว ซึ่งช่วยให้คุณครูสามารถประเมินความเสี่ยง บันทึก และแก้ไขข้อมูลนักเรียนข้ามอุปกรณ์ (คอมพิวเตอร์และโทรศัพท์มือถือ) ได้อย่างสะดวกรวดเร็วและอัปเดตตรงกันทันที 100%

สำรองและนำเข้าข้อมูลฉุกเฉิน

ใช้สำหรับนำเข้าข้อมูลรายชื่อจากตาราง CSV ในช่วงเริ่มต้น หรือเพื่อสำรองข้อมูลนักเรียนเก็บไว้ในเครื่องคอมพิวเตอร์ของคุณเพื่อความปลอดภัยทางข้อมูลสูงสุด

ลากและวางไฟล์ CSV หรือ คลิกเลือกไฟล์

เพื่อนำข้อมูลประวัตินักเรียนจำนวนมากเข้าสู่ฐานข้อมูล
**คำแนะนำ**: การนำเข้า CSV ระบบจะทำความเข้าใจโครงสร้างหัวตารางโดยอัตโนมัติ และข้อมูลจะถูกเขียนขึ้นฐานข้อมูล Supabase เมื่อกดบันทึกยืนยันข้อมูล
ระบบฐานข้อมูลคลาวด์ซิงค์สด (Supabase Cloud Sync)

เชื่อมต่อกับฐานข้อมูลความปลอดภัยสูงของ **Supabase (PostgreSQL)** เพื่อซิงค์ข้อมูลแก้ไขรายชื่อนักเรียนและประเมินความเสี่ยงระหว่างอุปกรณ์แบบเรียลไทม์ (PC และสมาร์ทโฟน) มั่นคงและไม่มีวันข้อมูลสูญหาย

⚪ โหมดปัจจุบัน: กำลังติดต่อฐานข้อมูล Supabase...
ระบบวิเคราะห์ประวัติกิจกรรมและข้อผิดพลาด (Diagnostics & Logs)

บันทึกประวัติกิจกรรมล่าสุดภายในเครื่องและดักจับข้อผิดพลาดเชิงลึกแบบอัตโนมัติ เพื่อช่วยระบุต้นตอปัญหาเน็ตหลุดหรือความขัดข้องของโครงสร้างชีตได้ทันที

⚪ ไม่มีประวัติกิจกรรมระบบในขณะนี้
บำรุงรักษาและสำรองข้อมูลระบบ

ฐานข้อมูลนักเรียนและการคัดกรองความเสี่ยงนี้ ถูกจัดเก็บอยู่ในระดับ LocalStorage ของเบราว์เซอร์เครื่องนี้ คุณสามารถดาวน์โหลดไฟล์สำรองข้อมูลเพื่อนำไปใช้ในเครื่องอื่นหรือนำเข้าคืนภายหลังได้

ส่งออกเป็นเอกสาร Excel/CSV

ดาวน์โหลดรายชื่อและข้อมูลคัดกรองทั้งหมดเป็นไฟล์ตารางสำหรับเปิดใน Excel

สำรองข้อมูลระบบเป็นไฟล์ JSON

สร้างสำเนาโครงสร้างฐานข้อมูลดิบทั้งหมดเพื่อใช้กู้คืนอย่างสมบูรณ์แบบ

กู้คืนฐานข้อมูลจากไฟล์สำรอง

เลือกไฟล์ JSON ของระบบที่ได้สำรองไว้มาทำการทับข้อมูลเดิม

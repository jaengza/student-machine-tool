# รายการงานระบบคลาวด์ซิงค์สด (Real-time Cloud Tasks)

- [x] เขียนและเตรียมสคริปต์ Google Apps Script (backend.js) สำหรับคุณครูนำไปติดตั้งหลัง Google Sheets
- [x] อัปเดตไฟล์ `index.html` เพื่อเพิ่มช่องป้อนลิงก์ Web App API ในส่วนหน้าตั้งค่าระบบ
- [x] อัปเดตไฟล์ `app.js` เพื่อเขียนคำสั่งซิงค์สดเรียลไทม์ โหมดออนไลน์ (Cloud Mode) โหลดประวัติและบันทึกประวัติกลับลง Sheets โดยตรง
- [x] อัปเดตไฟล์ `style.css` สำหรับการตกแต่งฟิลด์ตั้งค่าออนไลน์แบบพระเมียม
- [x] ส่งงานทั้งหมดขึ้นระบบ GitHub Repository และเผยแพร่ออนไลน์
- [x] เขียนคู่มือการติดตั้ง Google Apps Script อย่างละเอียดและเป็นขั้นตอนให้กับคุณครูในห้องแชท
- [x] แก้ไขบั๊กการซิงค์ข้อมูลด่วน (Quick Sync) ดึงข้อมูลไม่ครบสำหรับคนที่ไม่มีรูปภาพ และป้องกันข้อมูลกลุ่มเรียนพังเนื่องจากชื่อคอลัมน์ใน Sheet ซ้ำกัน
  - [x] วิเคราะห์บั๊กและดึงข้อมูล Google Sheet จริงมาตรวจสอบโครงสร้าง
  - [x] พัฒนา Smart Index Mapping Engine ใน `app.js` เพื่อแยกแยะคอลัมน์ที่ชื่อซ้ำกันโดยใช้ประเภทข้อมูล (URL vs Text)
  - [x] ปรับปรุงปุ่ม "ซิงค์ด่วน" บนหน้า Dashboard ให้ใช้ URL Google Sheet ล่าสุดที่ผู้ใช้กรอก
  - [x] อัปเกรดสถาปัตยกรรมเป็น Dual-source Database Merging (v7.0) ผสานไฟล์ 2 ตัวดึงรายชื่อหลัก 100 กว่าคนพร้อมเชื่อมโยงรูปถ่ายจากอีกไฟล์เข้าหากันอัตโนมัติ
  - [x] ยืนยันการทำงานของระบบและผลักดันโค้ดขึ้น GitHub
